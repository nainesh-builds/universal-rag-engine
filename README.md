# universal-rag-engine
A Retrieval-Augmented Generation (RAG) system capable of ingesting and processing documents in various formats (PDF, DOCX, TXT, etc.) and generating context-aware answers.

## Features
- Supports multiple document formats
- Automatically converts unsupported files
- Scalable ingestion from directories
- URL-based knowledge retrieval

## Getting Started
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Start the API server

## Tech Stack
- Python
- FAISS for vector search
- OpenAI GPT-3 / HuggingFace for LLMs
- FastAPI (for the backend)
